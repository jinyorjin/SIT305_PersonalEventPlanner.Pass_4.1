Personal Event Planner (Pass Task 4.1)
A simple Android app to manage your schedule, built with Kotlin and Room Database.

 Features
CRUD Operations: Add, View, Update, and Delete events.

Local Persistence: Data stays safe using Room DB.

Modern Navigation: Easy switching with Bottom Navigation Bar.

Smart Validation:

Prevents selecting past dates.

Ensures Title and Date are not empty.

Auto-Sorting: Upcoming events are automatically sorted by date.

Tech Stack
Language: Kotlin

Architecture: MVVM + ViewBinding

Database: Room (Standard 2026)

Navigation: Jetpack Navigation Component
